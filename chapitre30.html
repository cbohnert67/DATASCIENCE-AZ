Chapitre 30 : Web Scraping avec BeautifulSoup - Quand les données ne sont pas servies sur un plateau

Parfois, les données que vous convoitez sont là, visibles sur une page web, mais il n'existe aucune API pour vous permettre de les récupérer proprement. Faut-il abandonner ? Non ! Le web scraping (ou "moissonnage web") est la technique qui permet d'écrire un programme pour naviguer sur une page web et en extraire les informations automatiquement. C'est une compétence extrêmement puissante, mais qui doit être utilisée avec précaution et éthique. C'est l'art de collecter des données quand il n'y a pas de "menu" officiel.

Comprendre la structure d'une page web

Les bases du HTML

Une page web est un document structuré en HTML (HyperText Markup Language). Imaginez-le comme un squelette où chaque os est une "balise" qui définit un type de contenu. Pour extraire l'information, il faut apprendre à reconnaître les os importants.

Voici un exemple simple de code HTML qui représente une "citation" :

<div class="quote" id="quote-1">
    <p class="text">"Le monde est un livre et ceux qui ne voyagent pas n'en lisent qu'une page."</p>
    <footer>
        -- <span class="author">Saint Augustin</span>, 
        <a href="/author/Saint-Augustin">(à propos)</a>
    </footer>
</div>

Décortiquons ce plan :

  • Balises (Tags) : On voit plusieurs balises : `
    ` (un conteneur), `

    ` (un paragraphe), `

    ` (un pied de page), `` (pour styliser un bout de texte), et `` (un lien hypertexte).
  • Attributs : La balise `` a un attribut `href` qui contient l'URL `/author/Saint-Augustin`. C'est une information précieuse si on veut scraper le lien.
  • Classes et IDs :
    • Le conteneur `
      ` a une class `quote`. Il y aura probablement plusieurs `div` avec cette classe sur la page, une pour chaque citation. C'est parfait pour dire à notre script : "Trouve-moi tous les éléments de la classe `quote`".
    • Ce même `
      ` a aussi un id `quote-1`. Cet `id` est unique sur toute la page. Il est utile pour cibler cet élément très précisément.

Considérations éthiques et légales : La Règle d'Or

Le web scraping n'est pas un acte anodin. La règle d'or est simple : comportez-vous comme un humain respectueux, pas comme un robot agressif. Un robot peut faire des milliers de requêtes par minute, ce qui peut être perçu comme une attaque et faire tomber le site web.

1. Respecter le fichier `robots.txt`

C'est la première porte à laquelle il faut frapper. Ce fichier, accessible à la racine du site (ex: `https://www.lemonde.fr/robots.txt`), est un code de conduite pour les robots. Un exemple :

User-agent: *
# Autorise tout le monde

Disallow: /admin/
Disallow: /panier/
# Interdit l'accès aux dossiers /admin/ et /panier/

`User-agent: *` signifie que les règles s'appliquent à tous les robots. `Disallow` indique les chemins interdits. Ignorer ce fichier est très mal vu.

2. Ne pas surcharger les serveurs

Un humain clique sur une page toutes les quelques secondes. Un robot peut le faire des centaines de fois par seconde. Pour le serveur, cela ressemble à une attaque par déni de service (DDoS). Vous risquez de ralentir le site pour tout le monde et de voir votre adresse IP bannie. La solution est simple : faites une pause d'une seconde ou deux entre chaque requête. C'est ce qu'on appelle le "throttling".

import time
for page in pages_a_scraper:
    # ... code pour scraper la page ...
    time.sleep(1) # On attend 1 seconde avant la prochaine requête

3. Vérifier les conditions d'utilisation et ne pas scraper de données personnelles

Cherchez le lien "Conditions d'Utilisation" ou "Terms of Service" en bas de page. Certains sites interdisent explicitement le scraping automatisé de leur contenu. De plus, la collecte de données personnelles (noms, emails, etc.) est strictement réglementée, notamment par le RGPD en Europe. La prudence est de mise.

Extraire des données avec BeautifulSoup

Le processus se fait toujours en deux temps : `requests` télécharge le plan (le code HTML), et `BeautifulSoup` nous donne les outils pour naviguer à l'intérieur de ce plan. Analysons notre code d'exemple étape par étape.

Étape 1 : Télécharger le code HTML

On utilise `requests.get()` pour agir comme un navigateur et demander le contenu de la page au serveur. L'objet `response` contient tout ce que le serveur a renvoyé. `response.text` est ce qui nous intéresse : le code HTML brut, sous forme de chaîne de caractères.

url = 'http://quotes.toscrape.com/'
try:
    response = requests.get(url)
    response.raise_for_status()  # On vérifie que tout s'est bien passé (code 200)
    # ... suite ...
except requests.exceptions.RequestException as e:
    print(f"Erreur de connexion : {e}")

Étape 2 : Créer la "Soupe"

Le HTML dans `response.text` est juste du texte. Pour le rendre navigable, on le passe à BeautifulSoup. On crée un objet `soup` qui représente le HTML de manière structurée, comme un arbre généalogique. `'html.parser'` est simplement le moteur d'analyse intégré à Python.

    soup = BeautifulSoup(response.text, 'html.parser')

Étape 3 : Naviguer et extraire

C'est là que le "scraping" commence. On utilise les méthodes de `soup` pour trouver les éléments qui nous intéressent.

  • Trouver la première occurrence : `.find()`
    `soup.find('h1')` cherche la première balise `

    ` qu'il trouve dans le document. `.text` permet d'extraire uniquement le contenu textuel à l'intérieur de la balise. Pour être plus précis, `soup.find('span', class_='text')` cherche la première balise `` qui possède l'attribut `class="text"`. (On utilise `class_` avec un underscore car `class` est un mot-clé réservé en Python).

  • Trouver toutes les occurrences : `.find_all()`
    `soup.find_all('a', class_='tag')` ne s'arrête pas au premier résultat. Il parcourt tout le document et renvoie une liste de toutes les balises `    ` qui ont la classe `tag`. C'est pourquoi on peut ensuite itérer sur cette liste avec une boucle `for` pour extraire le texte de chaque tag un par un. 
    # Trouver le premier h1
    premier_titre = soup.find('h1').text
    print(f"Titre: {premier_titre}")

    # Trouver la première citation
    premiere_citation = soup.find('span', class_='text').text
    print(f"Citation: {premiere_citation}")

    # Trouver tous les tags
    tags = soup.find_all('a', class_='tag')
    for tag in tags:
        print(f"- {tag.text}")

Challenge pour vous !

Le site `quotes.toscrape.com` est spécialement conçu pour s'entraîner au scraping en toute légalité.

Votre mission :

Scrapez la page d'accueil de `quotes.toscrape.com` et récupérez les trois informations suivantes pour la toute première citation de la page : le texte complet, le nom de l'auteur, et la liste de ses "tags".

Cliquez ici pour la solution 
import requests
from bs4 import BeautifulSoup

url = 'http://quotes.toscrape.com/'
response = requests.get(url)
if response.status_code == 200:
    soup = BeautifulSoup(response.text, 'html.parser')
    
    first_quote_div = soup.find('div', class_='quote')
    
    text = first_quote_div.find('span', class_='text').text
    author = first_quote_div.find('small', class_='author').text
    tags_list = [tag.text for tag in first_quote_div.find_all('a', class_='tag')]

    print("--- Première Citation ---")
    print(f"Texte : {text}")
    print(f"Auteur : {author}")
    print(f"Tags : {tags_list}")
