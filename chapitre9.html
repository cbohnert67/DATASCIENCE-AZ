Chapitre 9. Introduction à la Visualisation de Données avec Matplotlib

Une image vaut mille lignes de données. Cette phrase n'a jamais été aussi vraie qu'en data science. Après avoir appris à résumer les données avec les statistiques, il est temps de les voir. La visualisation est l'un des outils les plus puissants du data scientist pour explorer un jeu de données, identifier des tendances, repérer des anomalies et, surtout, communiquer ses résultats de manière percutante. Matplotlib est la bibliothèque historique et fondamentale de l'écosystème Python. La maîtriser, c'est s'assurer de pouvoir, en toutes circonstances, transformer les nombres bruts en informations visuelles claires.

L'anatomie d'un graphique Matplotlib

Pour créer des graphiques efficaces, il faut connaître le vocabulaire. L'approche la plus puissante, dite "orientée objet", consiste à créer explicitement une Figure (la toile) et un ou plusieurs Axes (les graphiques). C'est sur ces objets que nous allons travailler.

[Image de l'anatomie d'un graphique Matplotlib]

1. La Figure et les Axes : La Toile et le Pinceau

Tout commence par la création de la toile (fig) et de la zone de dessin (ax). C'est la première ligne que vous écrirez presque toujours.

import matplotlib.pyplot as plt
import numpy as np

# Crée une Figure (la fenêtre) et un ensemble d'Axes (le graphique)
# figsize=(largeur, hauteur) en pouces
fig, ax = plt.subplots(figsize=(10, 6))

2. Le Traçage : Dessiner sur les Axes

Une fois que vous avez votre zone de dessin ax, vous utilisez ses méthodes (.plot(), .scatter(), etc.) pour y tracer vos données.

x = np.linspace(0, 10, 100)
y = np.sin(x)

# On DESSINE sur l'objet 'ax'
ax.plot(x, y, label='sin(x)')

3. La Personnalisation : Titres, Étiquettes et Légende

Un graphique sans texte est illisible. On utilise les méthodes .set_*() de l'objet ax pour ajouter des informations contextuelles.

# Ajouter un titre au graphique (Axes)
ax.set_title("Exemple de Graphique", fontsize=16)

# Nommer les axes
ax.set_xlabel("Axe des X", fontsize=12)
ax.set_ylabel("Axe des Y", fontsize=12)

# Ajouter une grille pour une meilleure lisibilité
ax.grid(True, linestyle='--', alpha=0.6)

# Afficher la légende (utilise le 'label' défini dans .plot())
ax.legend()

4. L'Affichage : Montrer le Résultat

Enfin, la commande plt.show() affiche la fenêtre contenant la Figure et tout ce que vous y avez dessiné.

# Afficher le graphique final
plt.show()

Un tour d'horizon des graphiques essentiels

Le choix du type de graphique dépend de la question que vous vous posez.

1. Graphique linéaire (ax.plot)

Idéal pour : Visualiser l'évolution d'une variable continue, souvent dans le temps.

Exemple : Évolution du cours d'une action, température au fil de la journée.

# Données : évolution sur 12 mois
mois = range(1, 13)
ventes = [120, 135, 150, 140, 165, 180, 175, 190, 210, 200, 220, 240]

fig, ax = plt.subplots()
ax.plot(mois, ventes, marker='o', linestyle='--', color='g')
ax.set_title("Évolution des Ventes Annuelles")
ax.set_xlabel("Mois")
ax.set_ylabel("Ventes (en milliers d'€)")
plt.show()

2. Nuage de points (ax.scatter)

Idéal pour : Visualiser la relation (ou l'absence de relation) entre deux variables numériques.

Exemple : Y a-t-il un lien entre la taille et le poids d'une personne ? Entre l'âge et le tarif payé sur le Titanic ?

import pandas as pd
df = pd.read_csv('train.csv')

fig, ax = plt.subplots(figsize=(10, 6))
# Utiliser la couleur (c) pour représenter une 3ème variable : la survie
scatter = ax.scatter(df['Age'], df['Fare'], alpha=0.5, c=df['Survived'])
ax.set_title("Relation Âge vs Tarif par Statut de Survie")
ax.set_xlabel("Âge")
ax.set_ylabel("Tarif")
ax.legend(*scatter.legend_elements(), title="Survie")
plt.show()

3. Diagramme à barres (ax.bar)

Idéal pour : Comparer des quantités entre différentes catégories.

Exemple : Ventes par pays, nombre de passagers par classe sur le Titanic.

# Préparer les données avec pandas
passagers_par_classe = df['Pclass'].value_counts().sort_index()

fig, ax = plt.subplots()
ax.bar(passagers_par_classe.index, passagers_par_classe.values, color=['#003f5c', '#7a5195', '#ef5675'])
ax.set_title("Nombre de passagers par classe")
ax.set_xlabel("Classe")
ax.set_ylabel("Nombre de passagers")
ax.set_xticks([1, 2, 3])
plt.show()

4. Histogramme (ax.hist)

Idéal pour : Visualiser la distribution d'une seule variable numérique en la regroupant en intervalles ("bins").

Exemple : Distribution des âges dans une population, distribution des notes à un examen.

fig, ax = plt.subplots()
# On retire les âges manquants (.dropna()) pour le graphique
ax.hist(df['Age'].dropna(), bins=20, edgecolor='black')
ax.set_title("Distribution de l'âge des passagers")
ax.set_xlabel("Âge")
ax.set_ylabel("Fréquence")
plt.show()

Challenge : Combiner des graphiques

La puissance de Matplotlib réside dans sa capacité à composer des visualisations complexes. Votre mission : créez une seule figure contenant deux sous-graphiques côte à côte. À gauche, tracez une onde sinusoïdale ; à droite, une onde cosinusoïdale. Donnez à chaque graphique un titre et des étiquettes d'axes.

Cliquez ici pour voir la solution 
import matplotlib.pyplot as plt
import numpy as np

# 1. Créer une Figure et un tableau d'Axes (1 ligne, 2 colonnes)
fig, axs = plt.subplots(1, 2, figsize=(12, 5))

# Données
x = np.linspace(0, 2 * np.pi, 200)
y_sin = np.sin(x)
y_cos = np.cos(x)

# 2. Travailler sur le premier sous-graphique
axs[0].plot(x, y_sin, color='blue')
axs[0].set_title("Onde Sinusoïdale")

# 3. Travailler sur le second sous-graphique
axs[1].plot(x, y_cos, color='green')
axs[1].set_title("Onde Cosinusoïdale")

plt.show()
