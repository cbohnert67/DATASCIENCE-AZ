Chapitre 8. Les Statistiques Descriptives - L'art de faire parler vos données

Avant de construire le moindre modèle d'intelligence artificielle ou la plus sophistiquée des visualisations, un bon data scientist fait une chose simple mais essentielle : il écoute ses données. Les statistiques descriptives sont les outils de cette première conversation. Elles nous permettent de résumer, de caractériser et de comprendre la structure d'un jeu de données sans chercher à en tirer des conclusions sur une population plus large. C'est l'art du premier contact, l'équivalent de la poignée de main avant une discussion approfondie. C'est ici que nous allons transformer une masse de chiffres intimidante en un résumé clair et concis.

Les outils de l'écoute

1. Mesures de tendance centrale : Où se situe le "centre" ?

Ces mesures nous donnent une idée de la valeur "typique" ou "centrale" de nos données.

La Moyenne (mean)

La somme de toutes les valeurs divisée par leur nombre. Très sensible aux valeurs extrêmes (aberrantes).

\[ \mu = \frac{1}{N} \sum_{i=1}^{N} x_i \] 

salaires = [30000, 32000, 35000, 40000, 250000]
df_salaires = pd.Series(salaires)
print(f"Moyenne : {df_salaires.mean()}") # Affiche 77400.0

La Médiane (median)

La valeur du milieu d'un ensemble de données triées. 50% des données sont en dessous, 50% sont au-dessus. Robuste aux valeurs extrêmes, elle est souvent plus représentative.

print(f"Médiane : {df_salaires.median()}") # Affiche 35000.0

Le Mode (mode)

La valeur qui apparaît le plus fréquemment. Particulièrement utile pour les données non numériques (catégorielles).

couleurs = ['bleu', 'noir', 'bleu', 'rouge', 'noir', 'bleu']
s_couleurs = pd.Series(couleurs)
print(f"Mode : {s_couleurs.mode()[0]}") # Affiche 'bleu'

2. Mesures de dispersion : À quel point les données sont-elles étalées ?

Il ne suffit pas de connaître le centre, il faut aussi savoir si les données sont regroupées ou très dispersées.

L'Écart-type (standard deviation)

Il représente la distance "moyenne" de chaque point par rapport à la moyenne. Un petit écart-type signifie que les données sont regroupées ; un grand écart-type signifie qu'elles sont étalées.

\[ \sigma = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2} \] 

print(f"Écart-type des salaires : {df_salaires.std():.2f}") # Affiche 95238.65

L'Écart Interquartile (IQR)

C'est l'étendue des 50% de données centrales, calculée comme la différence entre le 3ème quartile (Q3, 75ème percentile) et le 1er quartile (Q1, 25ème percentile). C'est une mesure de dispersion robuste.

\[ \text{IQR} = Q3 - Q1 \] 

Q1 = df_salaires.quantile(0.25)
Q3 = df_salaires.quantile(0.75)
IQR = Q3 - Q1
print(f"Écart Interquartile : {IQR}") # Affiche 8000.0

3. La Forme de la distribution

Une distribution nous montre la fréquence d'apparition de chaque valeur. On la visualise avec un histogramme pour voir si elle est symétrique (comme une courbe en cloche) ou asymétrique (avec une "queue" qui s'étend vers la droite ou la gauche).

import matplotlib.pyplot as plt
import numpy as np

# Données symétriques (loi normale)
data_sym = np.random.randn(1000)
pd.Series(data_sym).hist(bins=30)
plt.title('Distribution Symétrique')
plt.show()

# Données asymétriques à droite
data_asym = np.random.exponential(size=1000)
pd.Series(data_asym).hist(bins=30)
plt.title('Distribution Asymétrique à Droite')
plt.show()

La conversation avec Pandas

Pandas rend cette première conversation incroyablement simple et efficace. Utilisons le jeu de données du Titanic pour illustrer.

1. .describe() : Le résumé express

C'est la commande la plus utile pour commencer. En une seule ligne, elle calcule les statistiques clés pour toutes les colonnes numériques : nombre de valeurs non nulles (count), moyenne (mean), écart-type (std), minimum (min), quartiles (25%, 50%, 75%) et maximum (max).

import pandas as pd
df = pd.read_csv('train.csv') # Assurez-vous d'avoir ce fichier
print(df.describe())

2. Calculer des statistiques individuelles

Vous pouvez aussi calculer chaque mesure séparément. C'est utile pour comparer des valeurs spécifiques, comme la moyenne et la médiane, pour évaluer l'asymétrie.

age_moyen = df['Age'].mean()
age_median = df['Age'].median()
print(f"Âge moyen : {age_moyen:.2f}")
print(f"Âge médian : {age_median:.2f}") # Note: la moyenne est légèrement > à la médiane, suggérant une légère asymétrie à droite.

3. .value_counts() : Le roi des catégories

Pour les données non numériques (catégorielles), .value_counts() est parfait pour compter les occurrences de chaque catégorie. En ajoutant normalize=True, on obtient les pourcentages.

print("Nombre de passagers par classe :")
print(df['Pclass'].value_counts())

print("\\nPourcentage de passagers par classe :")
print(df['Pclass'].value_counts(normalize=True))

4. Visualiser les distributions

Un histogramme permet de voir la forme de la distribution d'une variable. Un box plot est excellent pour visualiser la médiane, les quartiles et détecter les valeurs aberrantes (outliers).

# Histogramme de la distribution de l'âge
df['Age'].hist(bins=20, grid=False, color='cyan')
plt.show()

# Box plot pour voir la dispersion du tarif ('Fare')
df.boxplot(column='Fare')
plt.show() # On voit beaucoup d'outliers (les points), des tarifs très élevés.

# La méthode la plus élégante est d'utiliser groupby()
description_par_groupe = df.groupby('Survived')[['Age', 'Fare']].describe()
print(description_par_groupe)

Analyse : La moyenne du tarif payé par les survivants est bien plus élevée. De plus, la moyenne d'âge des survivants est légèrement inférieure. Cela suggère que les passagers plus riches et plus jeunes avaient de meilleures chances de survie.
